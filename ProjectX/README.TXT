
PROJECT TITLE: The Battle of Dagobah

The Battle of Dagobah is a two-player battleship game featuring 
ewoks and wookies. Before running, update directory in ProjectX.java at lines 273,
290, and 346 to directory path where Original Planet.txt and Planet.txt are saved on user system.

To begin, run MyProjectX.java -- the planetary
information will be read in from the OriginalPlanet.txt file, and will
be saved to the Planet.txt file if the user chooses to save the game.
